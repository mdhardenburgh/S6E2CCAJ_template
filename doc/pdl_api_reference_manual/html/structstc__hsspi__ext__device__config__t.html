stc_hsspi_ext_device_config_t Struct Reference
High Speed Quad Serial Peripheral Interface (HSSPI) » Data Structures

Description

(High Speed) SPI external device configuration.

This struct is used to configure 1 of 4 possible devices by calling the HsSpi_SetExternalDeviceConfig() function.
