Parameters
[in]pstcIccPointer to ICC instance
[in]enIrqSelInterrupt selection
Return values
OkInterrupts has been enabled and callback fucntions are set
ErrorInvalidParameterIf one of following conditions are met:
  • pstcICC == NULL
  • pstcInt == NULL
  • pstcIntCb == NULL

boolean_t Icc_GetStatus ( volatile FM_SMCIF_TypeDef *  pstcIcc,
en_icc_status_t  enStatus 
)

Get status of ICC according to status type.

Parameters
[in]pstcIccPointer to ICC instance
[in]enStatusICC status type
  • IccTxEmpty Transmit buffer empty
  • IccRxFull Receive buffer full
  • IccRxAction Receive is in action
  • IccTxAction Transmit is in action
  • IccCardDetect Card insertion is detected
  • IccCardEvent Card event is detected
  • IccReceiveOverflow Receive buffer overflow
  • IccIdleTmrRun Idle timer running
  • IccRdFifoOverrun Read FIFO overflow
  • IccRdFifoFull Read FIFO full
  • IccWrFifoEmpty Write FIFO empty
  • IccTxResend Trasmit detect resend
  • IccRxResend Receive detect resend
Return values
FALSEThe event do not happen
TRUEThe event happen

en_icc_irq_sel_t  enIrqSel 
)

Get interrupt status of ICC according to status type.

Parameters
[in]pstcIccPointer to ICC instance
[in]enIrqSelICC interrupt type
  • IccRxFullIrq RX buffer full interrupt status
  • IccTxEmptyIrq TX buffer empty interrupt status
  • IccRxStartBitIrq RX start bit detect interrupt status
  • IccCardEventIrq Card event interrupt status
  • IccIdleTimerIrq Idle timer interrupt status
  • IccReadFifoIrq Read FIFO interrupt status
  • IccWriteFifoIrq Write FIFO interrupt status

Return values
FALSEThe interrupt status is not set
TRUEThe interrupt status is set
