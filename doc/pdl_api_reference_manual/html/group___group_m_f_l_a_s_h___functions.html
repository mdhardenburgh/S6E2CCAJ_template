en_result_t MFlash_ChipErase ( boolean_t  bCrRemain)

Flash chip erase.

Parameters
bCrRemainProtect CR data or not
  • TRUE Remain CR data after chip erase
  • FALSE CR data will be lost after chip erase
Returns
Operation status
Return values
OkChip erase successfully done.
ErrorInvalidParameterIf one of following conditions are met:
  • automatic algorithm of flash memory execution abnormally
