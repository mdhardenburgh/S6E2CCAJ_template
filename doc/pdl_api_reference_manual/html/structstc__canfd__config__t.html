stc_canfd_config_t Struct Reference
Controller Area Network with Flexible Data Rate(CANFD) » Data Structures

Description

CAN FD configuration.

Is used to set the CAN operation mode and CAN FD clock frequency. Optionally reception, transmission, error and status notification callback functions can be set.
