stc_can_msg_t Struct Reference
Controller Area Network (CAN) » Data Structures

Description

CAN message.

This structure stores a CAN message, including the identifier, data and data length code (DLC). It also contains an overflow and new flag which indicates the message state for received messages.

Data Fields

stc_can_msg_id_t stcIdentifier
 11- or 29-bit identifier (ID).
 
stc_can_msg_data_t stcData
 Data and DLC.
 
volatile boolean_t bOverflow
 TRUE if new message was received while bNew still TRUE.
 
volatile boolean_t bNew
 TRUE if new message received (must be reset to FALSE by application to release message object).
