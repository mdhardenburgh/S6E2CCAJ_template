enum en_hsspi_rx_status_t

High speed SPI reception status.

Status value used either in SpiRxStatus callback function or in HsSpi_GetDirectModeRxStatus() function (for direct mode polling operation).

Enumerator
HsSpiRxFinished 

Receive FIFO and shift register is empty.

HsSpiRxFifoLow 

Receive FIFO fill level is less or equal to threshold.

HsSpiRxNoStatusAvailable 

SPI instance is not valid, mode is set to command sequencer or reception is ongoing and FIFO level is higher than threshold.

enum en_hsspi_tx_status_t

High speed SPI transmission status.

Status value used either in SpiTxStatus callback function or in HsSpi_GetDirectModeTxStatus() function (for direct mode polling operation).

Enumerator
HsSpiModeError 

Illegal mode or parameter error.

HsSpiTxFinished 

Transmit FIFO and shift register is empty.

HsSpiTxFifoLow 

Transmit FIFO fill level is less or equal to threshold.

HsSpiTxNoStatusAvailable 

SPI instance is not valid, mode is set to command sequencer or transmission is ongoing and FIFO level is higher than threshold.

enum en_hsspi_fifo_width_t

Enumerator

8-bit FiFo width

HsSpiFifoWidth16 

16-bit FiFo width

HsSpiFifoWidth24 

24-bit FiFo width

HsSpiFifoWidth32 

32-bit FiFo width

enum en_hsspi_command_sequencer_control_t

High speed SPI command sequencer control.

This enum is used to configure the phases of the serial transaction generated by the command sequencer for memory read or write operations.

Enumerator
HsSpiCmdTransmitAddress07To00 

Transmit address bits 7:0 of serial memory address, DEC = 1.

HsSpiCmdTransmitAddress08To15 

Transmit address bits 15:8 of serial memory address, DEC = 1.

HsSpiCmdTransmitAddress16To23 

Transmit address bits 23:16 of serial memory address, DEC = 1.

HsSpiCmdTransmitAddress24To31 

Transmit address bits 31:24 of serial memory address, DEC = 1.

HsSpiCmdTransmitHighZStateByte 

High-Z byte, output signals are tri-stated for 1 byte time, DEC = 1.

HsSpiCmdTransmitHighZStateNibble 

High-Z nibble, output signals are tri-stated for 1 nibble time after 4 bits of the transmit data have been sent,DEC = 1.

HsSpiCmdTransmitData 

Data "as it is", no decoding, DEC = 0.

HsSpiCmdEndOfList 

End of list, DEC = 1.

Enumerator

HsSpiClkLowOutRisingInFalling

HsSpiClkHighOutRisingInFalling 

Serial clock high during idle, data output set with rising clock edge, data input sampling with falling clock edge.

HsSpiClkHighOutFallingInRising 

Serial clock high during idle, data output set with falling clock edge, data input sampling with rising clock edge.

HsSpiClkLowOutFallingInFalling 

Serial clock low during idle, data output set with falling clock edge, data input sampling with falling clock edge.

HsSpiClkLowOutRisingInRising 

Serial clock low during idle, data output set with rising clock edge, data input sampling with rising clock edge.

HsSpiClkHighOutRisingInRising 

Serial clock high during idle, data output set with rising clock edge, data input sampling with rising clock edge.

HsSpiClkHighOutFallingInFalling 

Serial clock high during idle, data output set with falling clock edge, data input sampling with falling clock edge.

enum en_hsspi_protocol_mode_t

SPI transfer mode (used for Command Sequencer and Direct mode)

Selects transfer protocol mode. The direct mode is active after call of HsSpi_Init, and can be changed by function HsSpi_SetMode(). RxTx-,RxOnly- and TxOnly modes will be selected automatically by the transfer functions.

Enumerator
HsSpiProtocolModeSingle 

Single mode.

enum en_hsspi_clk_delay_t

High Speed SPI clock delay.

Slave-Select to Clock Delay of Peripheral 0 (SS2CD[1:0]). This bit is used only when HS-SPI is configured as SPI master in direct mode or in command sequencer mode. It defines a setup time for the slave device. By delaying the toggling of SCLK, SPI delays the data transmission (of slave) from the chip select active edge by a multiple of SCLK cycles.

If SPIn_PCC0~3:CPHA = '0', the delay between assertion of slave select and first edge on the SCLK is given by: (SS2CD + 0.5) number of clock periods of SCLK. If SPIn_PCC0~3:CPHA = '1', the delay between assertion of slave select and first edge on the SCLK is given by: (SS2CD) number of clock periods of SCLK.

When the slave select becomes active, the slave has to prepare data transfer within the delay time de?ned by SS2CD bits.
