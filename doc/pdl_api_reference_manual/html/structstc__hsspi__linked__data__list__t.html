stc_hsspi_linked_data_list_t Struct Reference
High Speed Quad Serial Peripheral Interface (HSSPI) » Data Structures

Description

(High Speed) SPI linked data list.

Used to connect various different buffer sources that shall be transmitted during the same transaction, e.g. Ethernet packet with different buffers for for the ISO/OSI layer headers (external Ethernet MAC connected via HS-SPI).
