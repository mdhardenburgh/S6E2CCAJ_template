en_result_t Mft_Frt_SetOffsetMode ( volatile FM_MFT_FRT_TypeDef *  pstcMft,
uint8_t  u8Ch,
boolean_t  bOffsetMode 
)

Configure FRT channel to offest mode or normall mode.

The FRT channel 1 or 2, set to offset mode, will be working with FRT channel 0 with offset mode. The channel with offset mode will follow the behavior FRT channel 0, thus it is no need to call Mft_Frt_Init() to initialize this channel. See peripheral manual about offset mode for the details.

Parameters
[in]pstcMftPointer to FRT instance
  • the pointer of FRT register structure
[in]u8ChChannel of Free run timer
  • FrtOffsetCh1 FRT channel 1 which can be set to offset mode
  • FrtOffsetCh2 FRT channel 2 which can be set to offset mode
[in]bOffsetMode
  • TRUE Set the FRT channel to Offset mode
  • FALSE Set the FRT channel to normal mode
Return values
OkMode has been set normally
ErrorInvalidParameterIf oen of following cases match:
  • pstcMft == NULL
  • enCh out of range
Note
Only TYPE3 of FM4 or later products has this function.
