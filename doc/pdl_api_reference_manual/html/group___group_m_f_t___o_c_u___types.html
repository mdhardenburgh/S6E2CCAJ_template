enum en_ocu_compare_mode_t

The compare mode of OCU couple channels (FM3 mode)

Note
The OCU channel with 1 change mode or 2 change mode must connect with FRT with up count mode. The OCU channel with active high mode or active low mode must connect with FRT with up/down count mode.
Enumerator
OcuOdd1ChangeEven1Change 

Odd OCU channel is set to 1 change mode, even OCU channel is set to 1 change mode.

OcuOdd2ChangeEven1Change 

Odd OCU channel is set to 2 change mode, even OCU channel is set to 1 change mode.

OcuOdd1ChangeEvenActiveHigh 

Odd OCU channel is set to 1 change mode, even OCU channel is set to active high mode.

OcuOddActiveHighEven1Change 

Odd OCU channel is set to active high mode, even OCU channel is set to 1 change mode.

OcuOddActiveLowEven1Change 

Odd OCU channel is set to active low mode, even OCU channel is set to 1 change mode.

OcuOddActiveHighEvenActiveHigh 

Odd OCU channel is set to active high mode, even OCU channel is set to active high mode.

OcuOddActiveLowEvenActiveLow 

Odd OCU channel is set to active low mode, even OCU channel is set to active low mode.
