enum en_dstc_cmd_t

DSTC Commands.

Enumerator
CmdStandyRelease 

Instructs DSTC to return from standby state into normal state.

CmdStandyTransition 

Instructs DSTC to standby state.

CmdSwclr 

Clears SWTR:SWST to '0'; negates SWINT interrupt signal.

CmdErclr 

Clears MONERS:EST, MONERS:DER, and MONERS:ESTOP.

CmdRbclr 

Clears DESP to which DSTP refers in previous transfer; Sets next DES; Clears HWDESP[n].

CmdMkclr 

Clears all DQMSK[n].
