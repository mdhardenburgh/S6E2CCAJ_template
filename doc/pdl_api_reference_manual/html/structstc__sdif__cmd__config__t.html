Data Fields

en_sdif_cmd_type_t enCmdType
 Command type.
 
boolean_t bDataPresent
 TRUE: Data is present and shall be transferred using the DAT line, FALSE: Commands using only CMD line.
 
boolean_t bCmdIndexCheck
 TRUE: Host Controller shall check the Index field in the response to see if it has the same value as the command index. If it is not, it is reported as a Command Index Error, FALSE: the Index field is not checked.
 
boolean_t bCmdCrcCheck
 TRUE: Host Controller shall check the CRC field in the response. If an error is detected, it is reported as a Command CRC Error. FALSE: the CRC field is not checked. The position of CRC field is determined according to the length of the response.
