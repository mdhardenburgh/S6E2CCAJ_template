stc_hsspi_command_sequencer_control_t Struct Reference
High Speed Quad Serial Peripheral Interface (HSSPI) » Data Structures

Description

(High Speed) SPI command sequencer control command.

Used to configure the command sequence with defined parameters or with user defined data values (by setting enCommand to HsSpiCmdTransmitData).
