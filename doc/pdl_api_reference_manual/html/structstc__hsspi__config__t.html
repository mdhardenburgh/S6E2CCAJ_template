stc_hsspi_config_t Struct Reference
High Speed Quad Serial Peripheral Interface (HSSPI) » Data Structures

Description

(High speed) SPI configuration.

Struct used for initialisation of SSPI module in direct mode, direct and command sequence related parameter can be configured at once, afterwards it is possible to switch between modes with HsSpi_SetMode().
