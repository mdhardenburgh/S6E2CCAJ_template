enum en_lpm_main_osc_t

Main oscillator selection types.

Enumerator
LpmMainOsc4M 

Support 4MHz main oscillator in deep standby mode, has the lowest power consumption.

LpmMainOsc4M8M 

Support 4MHz and 8MHz main oscillator in deep standby mode, has the middle power consumption.

LpmMainOsc4M8M16M 

Support 4MHz, 8MHz, and 16MHz main oscillator in deep standby mode, has higher power consumption.

LpmMainOsc48M 

Support 48MHz main oscillator in deep standby mode, has the highest power consumption.
