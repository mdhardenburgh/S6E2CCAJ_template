enum en_adcmp_buf_t

Define Adcmp buffer type.

Enumerator
AdcmpBufDisable 

disable Adcmp buffer function

AdcmpBufFrtZero 

transfer buffer of ACMP and ACMC when counter value of Frt connected= 0x0000

AdcmpBufFrtPeak 

transfer buffer of ACMP and ACMC when counter value of Frt connected= TCCP

AdcmpBufFrtZeroPeak 

transfer buffer of ACMP and ACMC both when counter value of Frt connected= 0x0000 and TCCP

AdcmpBufFrtZeroMszcZero 

transfer buffer of ACMP and ACMC when counter value of Frt connected= 0x0000 and zero value detection mask counter value is 0

AdcmpBufFrtPeakMspcZero 

transfer buffer of ACMP and ACMC when counter value of Frt connected= peak and peak value detection mask counter value is 0

AdcmpBufFrtZeroMszcZeroOrFrtPeakMspcZero 

transfer buffer of ACMP and ACMC when counter value of Frt connected= 0x0000 and zero value detection mask counter value is 0 or counter value of Frt connected= peak and peak value detection mask counter value is 0
